GENERAL USAGE NOTES
--------------------

- Before disable the user of GotoMeeting you should download 2 libraries. Newtonsoft.json.dll and CitrixGotoUserAPI.dll from        https://developer.citrixonline.com/code-samples . Open the link and click on the CitrixGoToProductsUserAPI.zip. Now extract both the    dlls and put both the dlls at the same location from where you want to run the script. You should Unblock the dll by clicking right  click on dll and then click on the properties menu then click on the Unblock button.

- You can directly mention the values of args[0], args[1] and args[2] in the sigle quote ('') and run the script.
