GENERAL USAGE NOTES
--------------------

- To get API and APP key you need to login into Datadog first. Now you have to click on the "Integrations" left side menu and then click on the APIs. Here you will see the API and APP key. You can copy these keys and paste it into the script.

- Now you have to pass the user's email id as an argument to enable the user.
