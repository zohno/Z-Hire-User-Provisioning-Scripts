GENERAL USAGE NOTES
--------------------

- To run this script you should mention the user name, password and server name of the webex domain in the script in the signle quote('').

- You have to send the SamAccountName as an argument to enable the user.

- To get the Site ID of your domain you should login in the server first then click on the Site Information (Home Icon). After it will display the Site ID. Copy that Site ID and paste it in the script.
