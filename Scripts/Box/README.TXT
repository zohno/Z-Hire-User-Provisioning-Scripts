GENERAL USAGE NOTES
--------------------

- To get Token key you need to login into http://box-token-generator.herokuapp.com/ first and generate the access token.

- Now go to https://app.box.com/developers/services and create a app from any name and then go to configuration tab and the application scopes you have to checked the Manage Users Check box.

- Now you have to pass the user name as an argument to enable the user.
