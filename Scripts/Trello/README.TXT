GENERAL USAGE NOTES
--------------------

- To get the access token and Api key you should go to https://developers.trello.com/get-started/start-building and click on the "Get Your Application Key" button. This link will provide you both the token and Api key. Copy these keys into the script.

- Now you can enter the name of your organization to which you want to add the member.

- You should pass the user name of the member as an argument.
